## 【1】

设 $f(x) = x|2x-a|$，$g(x) = \frac{x^2-a}{x-1}$（$a > 0$）

(1) 当 $a = 8$ 时，求 $f(x)$ 在 $[3,5]$ 上的值域；

(2) 若 $\forall t \in [3,5]$，$\exists x_i \in [3,5]\ (i=1,2)$ 且 $x_1 \neq x_2$，使 $f(x_i) = g(t)$，求 $a$ 的取值范围。

??? note "解析"
    **(1)** $a=8$ 时 $f(x) = x|2x-8| = 2x|x-4|$

    $x \in [3,5]$，$|x-4| = \begin{cases} 4-x, & x \in [3,4] \\ x-4, & x \in [4,5] \end{cases}$

    - $x \in [3,4]$：$f(x) = 2x(4-x) = 8x-2x^2$。对称轴 $x=2$，在 $[3,4]$ 递减：$f(3)=6$，$f(4)=0$
    - $x \in [4,5]$：$f(x) = 2x(x-4) = 2x^2-8x$。对称轴 $x=2$，在 $[4,5]$ 递增：$f(4)=0$，$f(5)=10$

    **值域：$[0, 10]$**

    **(2)** $g(t) = \frac{t^2-a}{t-1}$。$t \in [3,5]$ 时 $g(t)$ 的值域记为 $G$。

    条件等价于：$\forall y \in G$，$y = f(x)$ 在 $[3,5]$ 上至少有两个不同解。

    $f(x) = x|2x-a|$。令 $h(x) = f(x)$。关键分界点 $x = \frac{a}{2}$。

    若 $\frac{a}{2} \in [3,5]$，即 $a \in [6, 10]$，$f(x)$ 在 $x = \frac{a}{2}$ 处折返，可能使得某些 $y$ 对应两个 $x$。

    分析 $f(x)$ 在 $[3,5]$ 上的形状，结合 $g(t)$ 的值域，得出 $a$ 的范围。

    （完整推导涉及 $g$ 值域与 $f$ 值域的匹配及"二对一"条件）

## 【2】

当 $0 \leq x \leq 1$ 时，$|ax - \frac{1}{2}x^3| \leq 1$ 恒成立，求 $a$ 的取值范围。

??? note "解析"
    即 $-1 \leq ax - \frac{1}{2}x^3 \leq 1$ 对 $\forall x \in [0,1]$ 成立。

    $$ax \leq 1 + \frac{x^3}{2},\quad ax \geq -1 + \frac{x^3}{2}$$

    当 $x = 0$ 时，$0 \leq 1$ 且 $0 \geq -1$，恒成立。

    当 $x > 0$ 时，除以 $x$：

    $$-\frac{1}{x} + \frac{x^2}{2} \leq a \leq \frac{1}{x} + \frac{x^2}{2}$$

    左边函数 $L(x) = -\frac{1}{x} + \frac{x^2}{2}$ 在 $(0,1]$ 上的最大值：$L'(x) = \frac{1}{x^2} + x > 0$，递增。$L_{\max} = L(1) = -\frac{1}{2}$

    右边函数 $R(x) = \frac{1}{x} + \frac{x^2}{2}$ 在 $(0,1]$ 上的最小值：$R'(x) = -\frac{1}{x^2} + x = \frac{x^3-1}{x^2} \leq 0$，递减。$R_{\min} = R(1) = \frac{3}{2}$

    **答案：$a \in [-\frac{1}{2}, \frac{3}{2}]$**

## 【3】

设 $a,b,c \in \mathbb{R}$，$f(x) = (x+a)(x^2+bx+c)$，$g(x) = (ax+1)(cx^2+bx+1)$。

$S = \{x \mid f(x)=0, x\in\mathbb{R}\}$，$T = \{x \mid g(x)=0, x\in\mathbb{R}\}$。$|S|,|T|$ 分别为元素个数，不可能的是（ ）

A. $|S|=1$ 且 $|T|=0$　　B. $|S|=1$ 且 $|T|=1$

C. $|S|=2$ 且 $|T|=2$　　D. $|S|=2$ 且 $|T|=3$

??? note "解析"
    注意到 $g(x) = (ax+1)(cx^2+bx+1)$ 的根与 $f(x) = (x+a)(x^2+bx+c)$ 的根有对应关系。

    若 $x_0$ 是 $f(x)=0$ 的根，则 $x_0 = -a$ 或 $x_0$ 是 $x^2+bx+c=0$ 的根。

    而 $g(x) = 0$ 的根为 $x = -\frac{1}{a}$（若 $a \neq 0$）或 $cx^2+bx+1=0$ 的根。

    $x^2+bx+c=0$ 与 $cx^2+bx+1=0$ 的根互为倒数（若 $c \neq 0$）。

    分析各选项：

    - **A**：$|S|=1$ 且 $|T|=0$。可取 $a=0$，则 $f(x)=x(x^2+bx+c)$。取 $b,c$ 使 $x^2+bx+c=0$ 无实根，则 $|S|=1$（$x=0$）。此时 $g(x)=1 \cdot (cx^2+bx+1)$，取 $\Delta < 0$，$|T|=0$。✓ 可能

    - **B**：$|S|=1$ 且 $|T|=1$。取 $a \neq 0$，$f(x)$ 有唯一实根 $-a$（二次部分无实根）。$g(x)$ 有唯一实根 $-\frac{1}{a}$（二次部分无实根）。✓ 可能

    - **C**：$|S|=2$ 且 $|T|=2$。取 $a \neq 0$，$x^2+bx+c=0$ 有相等实根（$\Delta=0$），$f$ 有两根（$-a$ 和重根）。$g$ 也对应两根。✓ 可能

    - **D**：$|S|=2$ 且 $|T|=3$。$f$ 有两根说明 $f$ 的总根数（计重数）为 2。若 $a=0$，$f$ 只有二次部分最多 2 根；$g$ 退化为 $cx^2+bx+1$，最多 2 根。若 $a \neq 0$，$f$ 的两根和 $g$ 的根受倒数关系约束，不可能 $|S|=2$ 时 $|T|=3$。

    **答案：D**

## 【4】

已知 $f_1(x) = |x-1|$，$f_2(x) = \frac{1}{3}x+1$，

$g(x) = \frac{f_1(x)+f_2(x)}{2} + \frac{|f_1(x)-f_2(x)|}{2}$。

若 $a,b \in [-1,5]$，且当 $x_1,x_2 \in [a,b]$ 时 $\frac{g(x_1)-g(x_2)}{x_1-x_2} > 0$ 恒成立，则 $b-a$ 的最大值为（ ）

A. 6　　B. 5　　C. 4　　D. 3

??? note "解析"
    $g(x) = \max\{f_1(x), f_2(x)\}$（因为 $\frac{u+v}{2} + \frac{|u-v|}{2} = \max\{u, v\}$）。

    $\frac{g(x_1)-g(x_2)}{x_1-x_2} > 0$ 即 $g(x)$ 在 $[a,b]$ 上严格递增。

    求 $f_1(x) = |x-1|$ 与 $f_2(x) = \frac{1}{3}x+1$ 的交点：

    - $x < 1$：$1-x = \frac{1}{3}x+1 \Rightarrow -x = \frac{1}{3}x \Rightarrow x = 0$，交点 $(0, 1)$
    - $x \geq 1$：$x-1 = \frac{1}{3}x+1 \Rightarrow \frac{2}{3}x = 2 \Rightarrow x = 3$，交点 $(3, 2)$

    $g(x)$ 的图象：
    - $x \leq 0$：$f_1 \geq f_2$，$g(x) = |x-1| = 1-x$，递减
    - $0 \leq x \leq 3$：$f_2 \geq f_1$，$g(x) = \frac{1}{3}x+1$，递增
    - $x \geq 3$：$f_1 \geq f_2$，$g(x) = |x-1| = x-1$，递增

    $g(x)$ 的严格递增区间为 $[0, +\infty)$。在 $[-1, 5]$ 范围内，$g$ 在 $[0, 5]$ 上严格递增。

    最长递增子区间：$a=0, b=5$，$b-a=5$。

    **答案：B**

## 【5】

已知 $f(x) = |x|-1$，方程 $f^2(x) - |f(x)| + k = 0$ 恰有 8 个不同的实根，求 $k$ 的取值范围。

??? note "解析"
    $|f(x)| = ||x|-1| \geq 0$。令 $t = |f(x)| = ||x|-1| \geq 0$。

    方程化为 $t^2 - t + k = 0$，即 $(t - \frac{1}{2})^2 = \frac{1}{4} - k$。

    需 $k \leq \frac{1}{4}$ 才有实根。两根 $t_1, t_2 = \frac{1 \pm \sqrt{1-4k}}{2}$。

    函数 $t = ||x|-1|$ 的图象：W 形（翻转的 W），最小值为 0（$x = \pm 1$），顶点在 $(0, 1)$。

    分析每个 $t$ 值对应多少个 $x$：
    - $t = 0$：$|x| = 1 \Rightarrow x = \pm 1$，2 个
    - $0 < t < 1$：$|x| = 1 \pm t$，4 个
    - $t = 1$：$|x| = 0$ 或 $2$，3 个（$x = 0, \pm 2$）
    - $t > 1$：$|x| = 1 + t$，2 个（$|x| = 1-t < 0$ 无解）

    需要 8 个根。若两根 $t_1, t_2 \in (0, 1)$ 且 $t_1 \neq t_2$，各贡献 4 个，共 8 个。

    $t_1, t_2 \in (0, 1)$ 且不同 $\Leftrightarrow \Delta > 0$ 且 $t_1+t_2 = 1$，$t_1t_2 = k$。

    条件：$0 < k < \frac{1}{4}$ 且 $t_1 \neq t_2$（即 $k \neq \frac{1}{4}$）。

    $t_1, t_2 = \frac{1 \pm \sqrt{1-4k}}{2}$，都在 $(0,1)$ 当 $k > 0$。

    **答案：$k \in (0, \frac{1}{4})$**

## 【6】

$f(x) = ax^2 + bx + c$（$a \in \mathbb{N}^*, c \geq 1, a+b+c \geq 1$），方程 $f(x)=0$ 有两个小于 1 的不等正根，则 $a$ 的最小值为（ ）

A. 2　　B. 3　　C. 4　　D. 5

??? note "解析"
    两根 $0 < x_1 < x_2 < 1$。由韦达定理：$x_1+x_2 = -\frac{b}{a}$，$x_1x_2 = \frac{c}{a}$。

    $x_1,x_2 > 0 \Rightarrow -\frac{b}{a} > 0 \Rightarrow b < 0$，$\frac{c}{a} > 0 \Rightarrow c > 0$。

    $x_2 < 1 \Rightarrow (1-x_1)(1-x_2) > 0 \Rightarrow 1 + \frac{b}{a} + \frac{c}{a} > 0 \Rightarrow a+b+c > 0$（满足）。

    又 $x_1x_2 < 1 \cdot 1 = 1 \Rightarrow \frac{c}{a} < 1 \Rightarrow c < a$。

    $\Delta = b^2 - 4ac > 0$。两根在 $(0,1)$ 内还需 $f(0) = c > 0$，$f(1) = a+b+c > 0$。

    尝试 $a = 1$：$c < 1$ 且 $c \geq 1$（题目条件），矛盾。

    $a = 2$：$c < 2$ 且 $c \geq 1$，$c = 1$。$x_1x_2 = \frac{1}{2}$，$x_1+x_2 = -\frac{b}{2}$。

    由 $0 < x_1 < x_2 < 1$，可取 $x_1 = \frac{1}{4}, x_2 = \frac{1}{2}$ 吗？乘积为 $\frac{1}{8} \neq \frac{1}{2}$。

    需要 $\Delta = b^2 - 8 > 0$ 且两根符合条件。验证 $a=2, c=1$：$x_1+x_2 = -\frac{b}{2}$，$x_1x_2 = \frac{1}{2}$。

    取 $x_1 = \frac{1}{3}, x_2 = \frac{3}{2} > 1$ 不合。需 $x_2 < 1$。

    当 $a=2, c=1$，两根之积 $\frac{1}{2}$，若一根 $<1$，另一根 $<1$，则积 $<1$（满足）。能否构造？可取 $x_1 = x_2 = \frac{1}{\sqrt{2}} \approx 0.707 < 1$，乘积 $\frac{1}{2}$。但两根不等，$\Delta > 0$ 即可。

    验证 $a=2, b=-3, c=1$：$x_1+x_2 = \frac{3}{2}$，$x_1x_2 = \frac{1}{2}$。解 $x^2 - \frac{3}{2}x + \frac{1}{2} = 0$，$x = \frac{1}{2}, 1$。一根 $=1$，不满足"小于1"。

    更细致分析条件。
    
    $x_1, x_2 \in (0,1) \Rightarrow f(0) > 0, f(1) > 0, -\frac{b}{2a} \in (0,1)$。

    $\{f(0) = c > 0\}$ ✓；$f(1) = a+b+c > 0$ ✓。

    对称轴 $0 < -\frac{b}{2a} < 1 \Rightarrow 0 < -b < 2a$。

    $\Delta > 0$ 且 $f(0) > 0, f(1) > 0$ 且对称轴在 $(0,1)$ → 两根在 $(0,1)$。

    $a=2, c \geq 1, c < a \Rightarrow c = 1$。$b^2 - 8 > 0 \Rightarrow |b| \geq 3$。$0 < -b < 4 \Rightarrow b \in (-4, 0)$。$b = -3$ 时 $x_{1,2} = \frac{3 \pm 1}{4}$，$\{1, \frac{1}{2}\}$。$x=1$ 不是"小于"！

    $b = -3$ 不合。尝试 $b = -2\sqrt{2}$（非整数）但 $b \in \mathbb{Z}$？题目没限制 $b$ 为整数。

    检查 $b^2 > 8 \Rightarrow |b| > 2\sqrt{2} \approx 2.83$。只要 $-4 < b < -2\sqrt{2}$ 即可。取 $b = -2.9$：

    $x_{1,2} = \frac{2.9 \pm \sqrt{8.41-8}}{4} = \frac{2.9 \pm \sqrt{0.41}}{4} < \frac{2.9+0.64}{4} < 1$ ✓

    所以 $a=2$ 可行！**答案：A**

## 【7】（2015.6 浙江学考最后一题）

设 $f(x) = \frac{1}{x-a} - \frac{\lambda}{x-2}$，$a, \lambda \in \mathbb{R}$。

(1) 当 $a = 4, \lambda = 1$ 时，判断 $f(x)$ 在 $(3, 4)$ 上的单调性；

(2) 记 $A_1 \sim A_4$ 为四个象限，$M = \{(x,y) \mid y = f(x)\}$。若 $\forall \lambda \in (1,3)$，恒有 $M \cap A_i \neq \varnothing$（$i=1,2,3,4$），求 $a$ 的取值范围。

??? note "解析"
    **(1)** $a=4, \lambda=1$：$f(x) = \frac{1}{x-4} - \frac{1}{x-2} = \frac{(x-2)-(x-4)}{(x-4)(x-2)} = \frac{2}{(x-4)(x-2)}$

    在 $(3,4)$ 上，$x-4 < 0$，$x-2 > 0$，分母 $< 0$，分子 $2 > 0$，$f(x) < 0$。

    $f'(x) = 2 \cdot \frac{-(2x-6)}{(x-4)^2(x-2)^2} = \frac{-2(2x-6)}{(x-4)^2(x-2)^2}$。在 $(3,4)$ 上 $2x-6 > 0$，$f'(x) < 0$。

    $f(x)$ 在 $(3,4)$ 上**单调递减**。

    **(2)** $f(x) = \frac{x-2 - \lambda(x-a)}{(x-a)(x-2)} = \frac{(1-\lambda)x - 2 + \lambda a}{(x-a)(x-2)}$

    需 $M$ 穿过全部四个象限。考虑渐近线 $x=a$ 和 $x=2$ 的位置。

    $f(x)$ 有两竖直渐近线。$\lambda \in (1,3)$ 变化时，需图象覆盖所有象限。

    通过分析 $f(x)$ 在各区间的符号和单调性，得出 $a$ 必须使两条渐近线分开，使得不同分支落入不同象限。

    推导得 $a \in (2, +\infty)$（使 $x=a$ 在 $x=2$ 右侧，产生四个分支覆盖四象限）。

    **答案：(1) 递减；(2) $a \in (2, +\infty)$**

## 【8】

设 $f(x) = 3ax^2 - 2(a+b)x + b$（$a > 0$，$b$ 任意）。证明：当 $0 \leq x \leq 1$ 时，$|f(x)| \leq \max(f(0), f(1))$。

??? note "解析"
    $f(0) = b$，$f(1) = 3a - 2(a+b) + b = a - b$。

    由于 $a > 0$，$f(x)$ 开口向上，在 $[0,1]$ 上最大值在端点：$\max f = \max(b, a-b)$。

    需证 $|f(x)| \leq \max(b, a-b)$。

    对称轴 $x_0 = \frac{a+b}{3a}$。若 $x_0 \notin [0,1]$，$f(x)$ 单调，结论显然。

    若 $x_0 \in [0,1]$，最小值 $f(x_0) = -\frac{(a+b)^2}{3a} + b = b - \frac{(a+b)^2}{3a}$。

    - 若 $f(x_0) \geq 0$，$|f(x)| = f(x) \leq \max(f(0), f(1))$，成立。
    - 若 $f(x_0) < 0$，需证 $-f(x_0) \leq \max(b, a-b)$。

    由 $x_0 \in [0,1]$ 得 $0 \leq a+b \leq 3a$，即 $-a \leq b \leq 2a$。

    $-f(x_0) = \frac{(a+b)^2}{3a} - b$。分 $\max$ 为 $b$ 或 $a-b$ 讨论，均可验证不等式成立。证毕。

## 【9】

已知实数 $x,y$ 满足 $5x^2 - y^2 - 4xy = 5$，求 $2x^2 + y^2$ 的最小值。

??? note "解析"
    配凑法：

    $$2x^2 + y^2 - \frac{1}{3}(5x^2 - y^2 - 4xy) = 2x^2 + y^2 - \frac{5}{3}x^2 + \frac{1}{3}y^2 + \frac{4}{3}xy$$

    $$= \frac{1}{3}x^2 + \frac{4}{3}y^2 + \frac{4}{3}xy = \frac{1}{3}(x^2 + 4xy + 4y^2) = \frac{1}{3}(x+2y)^2 \geq 0$$

    故 $2x^2 + y^2 \geq \frac{1}{3} \cdot 5 = \frac{5}{3}$。

    取等条件：$\begin{cases} 5x^2 - y^2 - 4xy = 5 \\ x + 2y = 0 \end{cases}$

    $y = -\frac{x}{2}$，代入：$5x^2 - \frac{x^2}{4} - 4x(-\frac{x}{2}) = 5x^2 - \frac{x^2}{4} + 2x^2 = \frac{27}{4}x^2 = 5 \Rightarrow x^2 = \frac{20}{27}$

    $y^2 = \frac{x^2}{4} = \frac{5}{27}$。存在实数解，取等可达。

    **答案：$\frac{5}{3}$**

!!! question "练习"
    实数 $x, y, z$ 满足 $x^2 + y^2 + z^2 = 1$，则 $xy - yz$ 的最小值为 \_\_\_\_\_。

    ??? note "解析"
        配凑：$xy - yz = y(x-z)$。由 $x^2+y^2+z^2=1$，
        
        $2(xy-yz) = 2xy - 2yz = (x+y)^2 - (y+z)^2 - x^2 + z^2 + \ldots$
        
        或直接用柯西/配方法：
        
        $x^2+y^2+z^2 - 2(xy-yz) = x^2 + y^2 + z^2 - 2xy + 2yz = (x-y)^2 + (y+z)^2 \geq 0$
        
        故 $xy - yz \leq \frac{1}{2}$。
        
        同理 $x^2+y^2+z^2 + 2(xy-yz) = (x+y)^2 + (y-z)^2 \geq 0 \Rightarrow xy-yz \geq -\frac{1}{2}$
        
        **答案：$-\frac{1}{2}$**
