# 1.4 指数函数

## 知识讲解

### 根式与指数幂

若 $x^n = a$，$x$ 叫 $a$ 的 $n$ 次方根（$n \in \mathbb{N}^*, n > 1$）。

- $n$ 为奇数：$\sqrt[n]{a^n} = a$；$n$ 为偶数：$\sqrt[n]{a^n} = |a|$
- $a^{-n} = \frac{1}{a^n}$，$a^{\frac{m}{n}} = \sqrt[n]{a^m}$（$a > 0$）

运算性质（$a>0, b>0, r,s\in\mathbb{R}$）：

$$a^r a^s = a^{r+s},\quad (a^r)^s = a^{rs},\quad (ab)^r = a^r b^r$$

### 指数函数

$y = a^x$（$a > 0, a \neq 1$），定义域 $\mathbb{R}$，值域 $(0, +\infty)$。

- $a > 1$ 递增，$0 < a < 1$ 递减，恒过 $(0, 1)$
- 直线 $x = 1$ 与各指数函数交点 $(1, a)$ 越高底数越大
- 本质：$f(x+y) = f(x)f(y)$ 且 $f(1) = a$ $\Rightarrow$ $f(x) = a^x$

## 【1】指数方程

关于 $x$ 的方程 $3 \cdot 16^x + 2 \cdot 81^x = 5 \cdot 36^x$ 的解为 \_\_\_\_\_。

??? note "解析"
    设 $u = 4^x > 0$，$v = 9^x > 0$，则 $16^x = u^2$，$81^x = v^2$，$36^x = uv$。

    原方程：$3u^2 + 2v^2 = 5uv$

    $$3u^2 - 5uv + 2v^2 = 0 \Rightarrow (3u - 2v)(u - v) = 0$$

    - 若 $u = v$：$4^x = 9^x \Rightarrow x = 0$
    - 若 $3u = 2v$：$3 \cdot 4^x = 2 \cdot 9^x \Rightarrow (\frac{4}{9})^x = \frac{2}{3} = (\frac{4}{9})^{\frac{1}{2}} \Rightarrow x = \frac{1}{2}$

    **答案：$x = 0$ 或 $x = \frac{1}{2}$**

!!! question "练习"
    方程 $3^x + 5^x + 11^x = 19^x \cdot \sqrt{x-1}$ 的实根个数为（ ）
    
    A. 0个　　B. 1个　　C. 2个　　D. 至少3个

    ??? note "解析"
        定义域 $x \geq 1$。
        
        令 $g(x) = \frac{3^x + 5^x + 11^x}{19^x} = \sqrt{x-1}$
        
        左边 $= (\frac{3}{19})^x + (\frac{5}{19})^x + (\frac{11}{19})^x$，各项底数 $< 1$，严格递减。
        右边 $= \sqrt{x-1}$，严格递增。
        
        $x = 1$：左边 $= \frac{3+5+11}{19} = 1$，右边 $= 0$，左边 > 右边。
        $x \to +\infty$：左边 $\to 0$，右边 $\to +\infty$。
        
        由单调性和介值定理，恰有一个交点。
        
        **答案：B**

## 【2】函数不等式

已知 $f(x) = x^5 + \frac{2^x - 1}{2^x + 1} + 1$，则 $f(2x-1) + f(x) > 2$ 的解集 \_\_\_\_\_。

??? note "解析"
    令 $h(x) = x^5 + \frac{2^x - 1}{2^x + 1}$。

    $x^5$ 是奇函数；$\frac{2^x - 1}{2^x + 1} = 1 - \frac{2}{2^x+1}$，$\frac{2^{-x} - 1}{2^{-x} + 1} = \frac{1-2^x}{1+2^x} = -\frac{2^x-1}{2^x+1}$，也是奇函数。

    $h(x)$ 是奇函数，且 $h'(x) = 5x^4 + \frac{2^{x+1}\ln 2}{(2^x+1)^2} > 0$，严格递增。

    $f(x) = h(x) + 1$。

    $$f(2x-1) + f(x) > 2 \Rightarrow h(2x-1) + 1 + h(x) + 1 > 2 \Rightarrow h(2x-1) + h(x) > 0$$

    $$h(2x-1) > -h(x) = h(-x) \quad (\text{奇函数})$$

    由 $h$ 严格递增：$2x-1 > -x \Rightarrow 3x > 1 \Rightarrow x > \frac{1}{3}$

    **答案：$(\frac{1}{3}, +\infty)$**

## 【3】唯一零点

已知 $f(x) = x^2 - 2x + a(e^{x-1} + e^{-x+1})$ 有唯一的零点，则 $a =$ \_\_\_\_\_。

??? note "解析"
    $e^{x-1} + e^{-(x-1)} \geq 2$（基本不等式），关于 $x = 1$ 对称。
    
    $x^2 - 2x = (x-1)^2 - 1 \geq -1$，也关于 $x = 1$ 对称。

    整个 $f(x)$ 关于 $x = 1$ 对称。若有唯一零点，必在对称轴 $x = 1$ 处。

    $$f(1) = 1 - 2 + a(e^0 + e^0) = -1 + 2a = 0 \Rightarrow a = \frac{1}{2}$$

    验证 $a = \frac{1}{2}$ 时确实唯一：$f(x) = (x-1)^2 - 1 + \frac{1}{2}(e^{x-1} + e^{-(x-1)})$

    令 $t = x-1$：$g(t) = t^2 - 1 + \frac{e^t + e^{-t}}{2} = t^2 + \cosh t - 1$

    $g(t) \geq 0^2 + 1 - 1 = 0$，当且仅当 $t = 0$ 取等。唯一零点。

    **答案：$a = \frac{1}{2}$**

## 【4】三角形三边条件

已知 $f(x) = \frac{9^x + k \cdot 3^x + 1}{9^x + 3^x + 1}$（$x \in \mathbb{R}$）

(1) 求值域；
(2) 若对任意实数 $x_1, x_2, x_3$，均存在以 $f(x_1), f(x_2), f(x_3)$ 为三边长的三角形，求 $k$ 的取值范围。

??? note "解析"
    **(1)** 令 $t = 3^x > 0$，$f = \frac{t^2 + kt + 1}{t^2 + t + 1} = 1 + \frac{(k-1)t}{t^2 + t + 1}$

    令 $g(t) = \frac{t}{t^2 + t + 1}$（$t > 0$）。$g(t)_{\max} = \frac{1}{3}$（$t = 1$ 时），$g(t) \to 0$（$t \to 0^+$ 或 $+\infty$）。

    $f = 1 + (k-1)g(t)$，值域：

    - $k > 1$：$f \in (1,\; 1+\frac{k-1}{3}]$
    - $k = 1$：$f = 1$
    - $k < 1$：$f \in [1+\frac{k-1}{3},\; 1)$

    **(2)** 三角形条件：任意两边之和大于第三边 $\Leftrightarrow$ $2\min f > \max f$。

    分 $k$ 讨论：

    - $k \geq 1$：$\min f$ 在 $g \to 0$ 时 $= 1$，$\max f = 1 + \frac{k-1}{3}$
    
    $2 \cdot 1 > 1 + \frac{k-1}{3} \Rightarrow k < 4$。又 $k \geq 1$，得 $1 \leq k < 4$

    - $k < 1$：$\min f = 1 + \frac{k-1}{3}$，$\max f$ 在 $g \to 0$ 时 $= 1$
    
    $2(1 + \frac{k-1}{3}) > 1 \Rightarrow 2 + \frac{2k-2}{3} > 1 \Rightarrow 2k-2 > -3 \Rightarrow k > -\frac{1}{2}$

    又 $k < 1$，得 $-\frac{1}{2} < k < 1$

    综上：**$k \in (-\frac{1}{2},\; 4)$**

!!! question "练习"
    $f(x) = \frac{e^x + m}{e^x + 1}$，若 $\forall a,b,c \in \mathbb{R}$ 都有 $f(a) + f(b) > f(c)$，求 $m$ 的范围。

    ??? note "解析"
        $f(x) = 1 + \frac{m-1}{e^x+1} \in (\min, \max)$。条件等价于 $2\min f > \max f$。分析得 **$m \in [\frac{1}{2}, 2]$**。

## 【5】指数方程组

若实数 $a, b, c$ 满足 $2^a + 2^b = 2^{a+b}$，$2^a + 2^b + 2^c = 2^{a+b+c}$，则 $c$ 的最大值是 \_\_\_\_\_。

??? note "解析"
    设 $x = 2^a, y = 2^b, z = 2^c$（$x, y, z > 0$）。

    $x + y = xy \Rightarrow xy = x + y \geq 2\sqrt{xy} \Rightarrow \sqrt{xy} \geq 2 \Rightarrow xy \geq 4$

    $x + y + z = xyz$，代入 $xy = x+y$：

    $xy + z = xyz \Rightarrow z(xy - 1) = xy \Rightarrow z = \frac{xy}{xy - 1} = 1 + \frac{1}{xy-1}$

    $xy \geq 4 \Rightarrow xy-1 \geq 3 \Rightarrow \frac{1}{xy-1} \leq \frac{1}{3} \Rightarrow z \leq \frac{4}{3}$

    $2^c \leq \frac{4}{3} \Rightarrow c \leq \log_2\frac{4}{3} = 2 - \log_2 3$

    **答案：$2 - \log_2 3$**

## 【6】抽象函数

设 $\mathbb{R}$ 上 $f(x)$ 满足：① $f(m+n) = f(m)f(n)$；② $f(0) \neq 0$；③ $x < 0$ 时 $f(x) > 1$。则正确结论的个数是（ ）

① $f(a) \cdot f(-a) = 1$　② $f(x)$ 在 $\mathbb{R}$ 上递减　③ 存在 $x_0$ 使 $f(x_0) < 0$　④ 若 $f(2) = \frac{1}{2}$，则 $f(\frac{1}{4}) = \frac{1}{4}, f(\frac{1}{6}) = \frac{1}{6}$

??? note "解析"
    由①：$f(0) = f(0)^2$，又 $f(0) \neq 0$，故 $f(0) = 1$。

    ① $f(a)f(-a) = f(0) = 1$ ✓

    ② $x_1 < x_2$ 时 $x_1-x_2 < 0$，$f(x_1-x_2) > 1$（条件③）。$f(x_1) = f(x_2)f(x_1-x_2) > f(x_2)$，递减 ✓

    ③ 若存在 $f(x_0) < 0$，则 $f(\frac{x_0}{2})^2 = f(x_0) < 0$，不可能。故 $f(x) > 0$ 恒成立 ×

    ④ $f(2) = f(1)^2 = \frac{1}{2} \Rightarrow f(1) = \frac{1}{\sqrt{2}}$

    $f(1) = f(\frac{1}{4})^4 \Rightarrow f(\frac{1}{4}) = (\frac{1}{\sqrt{2}})^{1/4} = 2^{-1/8} \neq \frac{1}{4}$ ×

    正确：①②，共 2 个。

    **答案：B**

## 【7】分式值域

求 $y = \frac{x^3 - x}{x^4 + 2x^2 + 1}$ 的值域。

??? note "解析"
    分母 $x^4 + 2x^2 + 1 = (x^2+1)^2$。分子 $x^3 - x = x(x^2-1)$。

    $$y = \frac{x(x^2-1)}{(x^2+1)^2}$$

    令 $t = x + \frac{1}{x}$（$|t| \geq 2$），则 $x^2+1 = x \cdot t$... 

    更直接：令 $x = \tan\theta$，则 $y = \frac{\tan\theta(\tan^2\theta-1)}{(\tan^2\theta+1)^2} = \frac{\tan\theta \cdot (-\cos 2\theta/\cos^2\theta)}{\sec^4\theta} = \ldots$

    换元 $t = x + \frac{1}{x}$：$x^2 + \frac{1}{x^2} = t^2 - 2$

    分子分母同除以 $x^2$：$y = \frac{x - \frac{1}{x}}{(x+\frac{1}{x})^2 \cdot x^{-2} \cdot x^2}?$

    重写：$y = \frac{x - \frac{1}{x}}{x^2 + 2 + \frac{1}{x^2}} = \frac{x - \frac{1}{x}}{(x + \frac{1}{x})^2}$

    令 $u = x - \frac{1}{x}$（$u \in \mathbb{R}$），$t = x + \frac{1}{x}$。注意 $t^2 - u^2 = 4$。

    当 $x \to 0$ 时分式需谨慎... 直接分析：

    $y = \frac{x(x^2-1)}{(x^2+1)^2}$，$y(-x) = -y(x)$，奇函数。$y(0) = 0$，$y(1) = 0$。

    求导得极值点 $x = \pm\sqrt{3}$ 时 $y = \pm\frac{\sqrt{3}}{8}$。

    值域：**$[-\frac{\sqrt{3}}{8}, \frac{\sqrt{3}}{8}]$**

!!! question "强基练习"
    求 $\frac{2x}{1+x^2} + \frac{6x}{1+9x^2}$ 的最大值。

    ??? note "解析"
        第一项 $\frac{2x}{1+x^2} \leq 1$（$x=1$ 取等），第二项 $\frac{6x}{1+9x^2} \leq 1$（$3x=1$ 取等）。两项不能同时取等。换元或求导得最大值。

## 【8】定义域与值域相同

若 $f(x) = m - \sqrt{x+3}$ 的定义域为 $[a, b]$，值域也为 $[a, b]$，求 $m$ 的取值范围。

??? note "解析"
    定义域 $x + 3 \geq 0 \Rightarrow x \geq -3$，故 $[a, b] \subseteq [-3, +\infty)$。

    值域 $[a, b] = \{m - \sqrt{x+3} \mid x \in [a, b]\}$。

    $g(x) = m - \sqrt{x+3}$ 在 $[a, b]$ 上严格递减（$g'(x) = -\frac{1}{2\sqrt{x+3}} < 0$）。

    故 $g(b) \leq g(x) \leq g(a)$，即 $[a, b] = [g(b), g(a)]$。

    $$\begin{cases} a = m - \sqrt{b+3} \\ b = m - \sqrt{a+3} \end{cases}$$

    两式相加：$a + b = 2m - \sqrt{a+3} - \sqrt{b+3}$

    两式相减：$a - b = \sqrt{a+3} - \sqrt{b+3}$

    由 $a - b = \sqrt{a+3} - \sqrt{b+3} \Rightarrow a + \sqrt{a+3} = b + \sqrt{b+3}$

    令 $h(t) = t + \sqrt{t+3}$（$t \geq -3$），$h'(t) = 1 + \frac{1}{2\sqrt{t+3}} > 0$，严格递增。

    $h(a) = h(b) \Rightarrow a = b$。但这样的话定义域退化为一点，值域也应是一点...

    实际上若 $a=b$，$f(a) = a \Rightarrow m - \sqrt{a+3} = a \Rightarrow m = a + \sqrt{a+3}$（$a \geq -3$）。

    此时 $m \geq -3 + 0 = -3$。

    若 $a \neq b$ 也成立？回到方程联立：

    $b = m - \sqrt{a+3}$，$a = m - \sqrt{b+3}$

    $\Rightarrow a + \sqrt{b+3} = b + \sqrt{a+3} \Rightarrow a - \sqrt{a+3} = b - \sqrt{b+3}$

    令 $\varphi(t) = t - \sqrt{t+3}$，$\varphi'(t) = 1 - \frac{1}{2\sqrt{t+3}}$。当 $t > -\frac{11}{4}$ 时 $\varphi' > 0$。

    $\varphi(a) = \varphi(b) \Rightarrow a = b$（在单调区间上）。

    综上，$a = b$。$m = a + \sqrt{a+3}$（$a \geq -3$）。

    **答案：$m \geq -3$（当 $a = -3$ 时 $m = -3$，$a \geq -3$ 时 $m \in [-3, +\infty)$）**
