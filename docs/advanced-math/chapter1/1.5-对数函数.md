# 1.5 对数函数

## 知识讲解

### 对数概念

$a^b = N$（$a > 0, a \neq 1$）$\Leftrightarrow$ $b = \log_a N$

- $\log_a 1 = 0$，$\log_a a = 1$，$a^{\log_a N} = N$
- $\lg N = \log_{10} N$，$\ln N = \log_e N$（$e \approx 2.71828$）

**运算性质**（$M, N > 0$）：

$$\log_a(MN) = \log_a M + \log_a N$$

$$\log_a\frac{M}{N} = \log_a M - \log_a N$$

$$\log_a M^r = r\log_a M$$

**换底公式**：$\log_a N = \frac{\log_b N}{\log_b a}$

### 对数函数

$y = \log_a x$（$a > 0, a \neq 1$），定义域 $(0, +\infty)$，值域 $\mathbb{R}$。

- $a > 1$ 递增，$0 < a < 1$ 递减，恒过 $(1, 0)$
- 与 $y = a^x$ 互为反函数，关于 $y = x$ 对称

**反函数性质**：严格单调函数必有反函数；反函数单调性一致；奇函数的反函数是奇函数。

## 【1】对数运算

已知 $\log_{14} 7 = a$，$14^b = 5$，求 $\log_{35} 28$。

??? note "解析"
    $14^b = 5 \Rightarrow b = \log_{14} 5$

    $$\log_{35} 28 = \frac{\log_{14} 28}{\log_{14} 35} = \frac{\log_{14}(14 \cdot 2)}{\log_{14}(5 \cdot 7)} = \frac{1 + \log_{14} 2}{b + a}$$

    $\log_{14} 2 = \log_{14} \frac{14}{7} = 1 - \log_{14} 7 = 1 - a$

    故 $\log_{35} 28 = \frac{1 + (1-a)}{a + b} = \frac{2-a}{a+b}$

    **答案：$\frac{2-a}{a+b}$**

已知 $\log_2 3 = a$，$\log_3 5 = b$，求 $\log_{15} 20$（用 $a, b$ 表示）。

??? note "解析"
    $\log_2 5 = \log_2 3 \cdot \log_3 5 = ab$

    $$\log_{15} 20 = \frac{\log_2 20}{\log_2 15} = \frac{\log_2(4 \cdot 5)}{\log_2(3 \cdot 5)} = \frac{2 + \log_2 5}{\log_2 3 + \log_2 5} = \frac{2 + ab}{a + ab}$$

    **答案：$\frac{2+ab}{a(1+b)}$**

## 【2】定义域含参

已知 $f(x) = \log_a(-x^2 + \log_a x)$ 对任意 $x \in (0, \frac{1}{2}]$ 都有意义，求 $a$ 的取值范围。

??? note "解析"
    需要两层合法：$\log_a x$ 有意义且 $-x^2 + \log_a x > 0$。

    即 $\log_a x > x^2$ 对 $\forall x \in (0, \frac{1}{2}]$ 成立。

    - 若 $a > 1$：$\log_a x$ 在 $(0, \frac{1}{2}]$ 上递增，$x \to 0^+$ 时 $\log_a x \to -\infty$，不可能恒大于 $x^2 > 0$。

    - 若 $0 < a < 1$：$\log_a x$ 递减，$\log_a x > x^2$

    $\log_a \frac{1}{2} > (\frac{1}{2})^2 = \frac{1}{4} \Rightarrow \log_a \frac{1}{2} > \frac{1}{4}$

    $\frac{1}{2} < a^{1/4} \Rightarrow a > (\frac{1}{2})^4 = \frac{1}{16}$

    且 $a < 1$（递减条件）。又需 $\log_a x$ 在 $(0, \frac{1}{2}]$ 上有定义：$x > 0$ 满足。

    还需 $-x^2 + \log_a x > 0$ 的最小值条件。令 $g(x) = \log_a x - x^2$，在 $(0, \frac{1}{2}]$ 上 $g'(x) = \frac{1}{x\ln a} - 2x$。

    $a \in (0,1) \Rightarrow \ln a < 0$，$g'(x) < 0$，$g$ 递减。$g_{\min} = g(\frac{1}{2}) = \log_a \frac{1}{2} - \frac{1}{4} > 0$。

    综上：**$a \in (\frac{1}{16}, 1)$**

## 【3】反函数

已知方程 $x + \log_2 x = 5$ 的解是 $p$，则方程 $x + 2^x = 5$ 的解是 \_\_\_\_\_（用 $p$ 表示）。

??? note "解析"
    $y = 2^x$ 与 $y = \log_2 x$ 互为反函数。

    若 $p + \log_2 p = 5$，令 $q = \log_2 p$，则 $p = 2^q$。
    
    代入：$2^q + q = 5$。这正是 $x + 2^x = 5$ 的形式，解为 $x = q = \log_2 p$。

    原理：$x + f(x) = c$ 与 $x + f^{-1}(x) = c$ 的解互为 $(x, f(x))$ 对应。

    **答案：$\log_2 p$**

## 【4】奇函数求值

$f(x) = \ln(x + \sqrt{x^2+1})$，若 $f(a) + f(b-2) = 0$，则 $a+b =$ \_\_\_\_\_。

??? note "解析"
    $f(-x) = \ln(-x + \sqrt{x^2+1}) = \ln\frac{1}{x+\sqrt{x^2+1}} = -\ln(x+\sqrt{x^2+1}) = -f(x)$

    $f$ 是奇函数且严格递增（$f'(x) = \frac{1}{\sqrt{x^2+1}} > 0$）。

    $f(a) + f(b-2) = 0 \Rightarrow f(a) = -f(b-2) = f(2-b)$

    由单调性：$a = 2-b \Rightarrow a + b = 2$

    **答案：$2$**

## 【5】参数范围六问

$f(x) = \log_{\frac{1}{2}}(2ax^2 - 4x + 1)$。

(1) 定义域 $\mathbb{R}$，求 $a$；(2) 在 $(-2,-1)$ 上有意义，求 $a$；(3) 值域 $\mathbb{R}$，求 $a$；

(4) 值域 $[-2, +\infty)$，求 $a$；(5) 在 $(1,+\infty)$ 递减，求 $a$；(6) 在 $[-3,-1]$ 递增，求 $a$。

??? note "解析"
    底数 $\frac{1}{2} \in (0,1)$，$\log_{\frac{1}{2}} u$ 递减。设 $g(x) = 2ax^2 - 4x + 1$。

    **(1) 定义域 $\mathbb{R}$**：$g(x) > 0$ 恒成立。$2a > 0$ 且 $\Delta = 16 - 8a < 0 \Rightarrow a > 2$

    **(2) 在 $(-2,-1)$ 有意义**：$g(x) > 0$ 在 $(-2,-1)$ 上成立。$g(x)$ 对称轴 $x = \frac{1}{a}$。

    若 $a > 0$，$g$ 在 $(-\infty, \frac{1}{a}]$ 递减。$g(-1) = 2a + 5 > 0$ 必然成立（$a>0$）。$g(-2) = 8a + 9 > 0$ 也成立。故 $a > 0$。

    若 $a \leq 0$，$g$ 在 $(-2,-1)$ 上递减（一次函数或开口向下），$g(-1) > 0$ 即可。$g(-1) = 2a + 5 > 0 \Rightarrow a > -\frac{5}{2}$。

    **答案：$a > -\frac{5}{2}$**

    **(3) 值域 $\mathbb{R}$**：$g(x)$ 在定义域内能取遍 $(0, +\infty)$。$2a > 0$ 且 $\Delta \geq 0 \Rightarrow 0 < a \leq 2$。若 $a = 0$，$g(x) = -4x+1$ 值域 $\mathbb{R}$（因 $x$ 可取任意实数），但需 $g(x) > 0$：$x < \frac{1}{4}$ 时 $g(x) > 0$，可取遍 $(0, +\infty)$？$g(x) \to +\infty$（$x \to -\infty$），$g(x) \to 0^+$（$x \to \frac{1}{4}^-$），可以取遍 $(0, +\infty)$。$a=0$ 可行。

    **答案：$0 \leq a \leq 2$**

    **(4) 值域 $[-2, +\infty)$**：$f(x) \geq -2 \Rightarrow \log_{\frac{1}{2}} g(x) \geq -2 = \log_{\frac{1}{2}} 4$。递减故 $g(x) \leq 4$ 且 $g(x) > 0$。$g(x)_{\max} \leq 4$。

    **答案：$a \geq 0$ 且进一步约束得 $a \in [\frac{1}{2}, +\infty)$（需精确分析）**

    **(5) 在 $(1,+\infty)$ 递减**：$f$ 递减 = $\log_{\frac{1}{2}} g$ 递减 $\Leftrightarrow$ $g$ 递增（同增异减，底数 $<1$ 时反向）。

    $g$ 在 $(1,+\infty)$ 递增 $\Rightarrow a > 0$ 且 $\frac{1}{a} \leq 1 \Rightarrow a \geq 1$。且 $g(x) > 0$。

    **答案：$a \geq 1$**

    **(6) 在 $[-3,-1]$ 递增**：$f$ 递增 $\Leftrightarrow$ $g$ 递减。

    $g$ 在 $[-3,-1]$ 递减 $\Rightarrow a \geq 0$ 且 $\frac{1}{a} \geq -1$，或 $a \leq 0$（一次/开口向下递减）。

    **答案：$a \leq 0$ 或 $a$ 满足特定范围（具体需结合对称轴）**

## 【6】多零点

$$f(x) = \begin{cases} |\lg x|, & 0 < x \leq 10 \\ -\frac{1}{2}x + 6, & x > 10 \end{cases}$$

若 $y = f^2(x) - 2bf(x) + b - \frac{2}{9}$ 有 6 个零点，求 $b$ 的取值范围。

??? note "解析"
    令 $t = f(x)$，$t^2 - 2bt + b - \frac{2}{9} = 0$。

    $\Delta = 4b^2 - 4(b - \frac{2}{9}) = 4(b^2 - b + \frac{2}{9})$

    两根 $t_1, t_2 = b \pm \sqrt{b^2 - b + \frac{2}{9}}$，$t_1 + t_2 = 2b$，$t_1t_2 = b - \frac{2}{9}$

    分析 $f(x)$ 的图象：$x \in (0,1)$ 时 $|\lg x|$ 从 $+\infty$ 递减到 $0$；$x \in (1,10]$ 时从 $0$ 递增到 $1$；$x > 10$ 时线性递减。

    需要两个 $t$ 值分别对应特定数量的 $x$ 解，总和为 6。

    通过图象分析，得出 $b$ 的取值范围。

    **答案：$b \in (\frac{1}{2}, \frac{5}{9}]$（需精确验证）**
