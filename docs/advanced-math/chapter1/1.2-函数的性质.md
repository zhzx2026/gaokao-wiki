# 1.2 函数的性质

## 知识讲解

### 函数的单调性

**定义**：$\forall x_1, x_2 \in I, x_1 < x_2$，若 $f(x_1) < f(x_2)$ 则 $f$ 在 $I$ 上单调递增；若 $f(x_1) > f(x_2)$ 则单调递减。

等价表述（递增）：

$$\forall x_1, x_2 \in I, x_1 \neq x_2, \quad \frac{f(x_1)-f(x_2)}{x_1-x_2} > 0$$

$$\forall x_1, x_2 \in I, x_1 \neq x_2, \quad (x_1-x_2)[f(x_1)-f(x_2)] > 0$$

**判断方法**：

1. **定义法**：对 $f(x_1) - f(x_2)$ 代数变形
2. **运算法则**：增+增=增，减+减=减；乘正常数不变，乘负常数反向
3. **复合函数**：**同增异减**——奇数个递减函数复合得递减，偶数个得递增

**复合函数单调性证明**：

已知 $y = f(x)$ 在 $\mathbb{R}$ 上递减，$y = g(x)$ 在 $\mathbb{R}$ 上递增，设 $h(x) = f(g(x))$。

取 $x_1 < x_2$，则 $g(x_1) < g(x_2)$（$g$ 递增），$f(g(x_1)) > f(g(x_2))$（$f$ 递减），故 $h(x_1) > h(x_2)$，$h(x)$ 递减。

### 函数的奇偶性

**定义**：

- 偶函数：$\forall x \in D, f(-x) = f(x)$，图象关于 $y$ 轴对称
- 奇函数：$\forall x \in D, f(-x) = -f(x)$，图象关于原点对称

前提：定义域关于原点对称。

**运算法则**：

- 奇±奇=奇，偶±偶=偶
- 奇×奇=偶，偶×偶=偶，奇×偶=奇
- 复合：有偶则偶；奇函数任意次复合仍为奇函数

### 函数的对称性

**轴对称**：$f(x)$ 关于 $x = a$ 对称 $\Leftrightarrow f(a+x) = f(a-x)$

等价于 $y = f(x+a)$ 是偶函数。偶函数是 $a=0$ 的特例。

**中心对称**：$f(x)$ 关于 $(a, b)$ 对称 $\Leftrightarrow f(a+x) + f(a-x) = 2b$

等价于 $y = f(x+a) - b$ 是奇函数。奇函数是 $(0,0)$ 的特例。

**对称位置的性质**：若 $f(x)$ 关于 $x = a$ 或 $(a, b)$ 对称，则定义域、单调区间、极值点、零点均关于 $a$ 对称。若 $\lim_{x \to +\infty} f(x)$ 存在：

$$\lim_{x\to +\infty} f(x) + \lim_{x\to -\infty} f(x) = 2b$$

## 【1】判断正确与否

(1) 函数 $f(x) = (x+2)\sqrt{\frac{x-2}{x+2}}$ 为偶函数；

(2) 函数 $f(x) = \frac{x+1}{x-1}$ 在 $\{x \in \mathbb{R} \mid x \neq 1\}$ 上单调递减；

(3) 如果定义在 $\mathbb{R}$ 上的函数 $f(x)$ 为奇函数，且在 $(0, +\infty)$ 上单调递增，则它在 $\mathbb{R}$ 上单调递增；

(4) 函数 $y = f(x+1)$ 为偶函数，则 $f(x+1) = f(-x-1)$；

(5) 函数 $y = f(x+1)$ 与函数 $y = f(1-x)$ 关于 $y$ 轴对称；

(6) 函数 $y = f(x)$ 的图象关于点 $(1, 2)$ 中心对称，则 $f(2-x) = 4 - f(x)$。

??? note "解析"
    **(1) ×** 定义域：$\frac{x-2}{x+2} \geq 0$ 且 $x+2 \neq 0$。解得 $x \in (-\infty, -2) \cup [2, +\infty)$。定义域不关于原点对称（不含 $(-2, 2)$ 的大部分），不是偶函数。

    **(2) ×** $f(x) = \frac{x+1}{x-1} = 1 + \frac{2}{x-1}$。求导 $f'(x) = -\frac{2}{(x-1)^2} < 0$（$x \neq 1$）。虽然 $f'(x) < 0$，但在 $(-\infty, 1)$ 和 $(1, +\infty)$ 上分别是递减的，整体不能说单调递减（在 $x=1$ 处断开，且 $x \to 1^-$ 时 $f(x) \to -\infty$，$x \to 1^+$ 时 $f(x) \to +\infty$）。严格说，该命题表述不严谨。

    **(3) ×** 反例：$f(x) = \begin{cases} x, & x > 0 \\ x, & x < 0 \end{cases}$ 但更准确的反例是奇函数在原点的行为。如 $f(x) = x$ 对 $x > 0$ 递增，对 $x < 0$ 也递增，但它仍是递增的。真正的反例：$f(x) = \begin{cases} x+1, & x>0 \\ 0, & x=0 \\ x-1, & x<0 \end{cases}$ 是奇函数，$(0, +\infty)$ 上递增，但在 $\mathbb{R}$ 上不递增（$f(-0.5) = -1.5$，$f(0.5) = 1.5$，不是 $-0.5 < 0.5 \nRightarrow f(-0.5) > f(0.5)$ 的问题...）。实际上奇函数在 $(0, +\infty)$ 递增不能推 $\mathbb{R}$ 上递增，反例需谨慎。但题中结论一般不成立，故为 ×。

    **(4) ×** $y = f(x+1)$ 是偶函数 $\Rightarrow f(-x+1) = f(x+1)$，即 $f(1-x) = f(1+x)$。题目写的是 $f(x+1) = f(-x-1)$，这是 $f(x+1) = f(-(x+1))$，即 $y = f(x)$ 是偶函数而非 $y = f(x+1)$。故命题错误。

    **(5) √** $y = f(x+1)$ 关于 $y$ 轴对称的点为 $(-x, f(x+1))$。该点在 $y = f(1-x)$ 上当且仅当 $f(1-(-x)) = f(1+x) = f(x+1)$，成立。两个函数的图象关于 $y$ 轴对称。

    **(6) √** 关于 $(1, 2)$ 中心对称 $\Leftrightarrow f(1+x) + f(1-x) = 4$。

    令 $t = 1-x$，则 $x = 1-t$：$f(2-t) + f(t) = 4$，即 $f(2-t) = 4 - f(t)$。

    即 $f(2-x) = 4 - f(x)$。✓

## 【2】偶函数"脱衣服"

已知 $f(x)$ 是定义在 $[-3, 3]$ 上的偶函数，且在 $[0, 3]$ 上单调递增，解不等式 $f(2x-1) > f(x)$。

??? note "解析"
    偶函数 $\Rightarrow f(-x) = f(x)$。$[0, 3]$ 上递增 $\Rightarrow$ $[-3, 0]$ 上递减（偶函数对称性）。

    不等式 $f(2x-1) > f(x)$，需要 $|2x-1| < 3$ 且 $|x| < 3$（自变量在定义域内）。

    由 $f$ 在 $[0, 3]$ 递增，在 $[-3, 0]$ 递减：

    $f(2x-1) > f(x) \Leftrightarrow |2x-1| > |x|$（偶函数，"脱衣服"——函数值大小取决于自变量绝对值大小）

    解 $|2x-1| > |x|$：平方得 $(2x-1)^2 > x^2 \Rightarrow 3x^2 - 4x + 1 > 0 \Rightarrow (3x-1)(x-1) > 0$

    $\Rightarrow x < \frac{1}{3}$ 或 $x > 1$

    还需满足定义域：$-3 \leq 2x-1 \leq 3 \Rightarrow -1 \leq x \leq 2$；$-3 \leq x \leq 3$。

    交集：$[-3, \frac{1}{3}) \cup (1, 2]$

    **答案：$x \in [-3, \frac{1}{3}) \cup (1, 2]$**

## 【3】复合函数单调区间

已知函数 $y = f(x)$ 在 $\mathbb{R}$ 上是减函数，则 $y = f(|x-3|)$ 的单调减区间是 \_\_\_\_\_。

??? note "解析"
    令 $u = |x-3| = \begin{cases} 3-x, & x < 3 \\ x-3, & x \geq 3 \end{cases}$

    $u$ 在 $(-\infty, 3]$ 递减，在 $[3, +\infty)$ 递增。

    $y = f(u)$，$f$ 在 $\mathbb{R}$ 上递减。

    由复合函数"同增异减"：

    - $x \in (-\infty, 3]$：$u(x)$ 递减，$f(u)$ 递减 → $y$ 递增
    - $x \in [3, +\infty)$：$u(x)$ 递增，$f(u)$ 递减 → $y$ 递减

    **答案：$[3, +\infty)$**

**变式**：如果改为 $y = f(x)$ 在 $(-\infty, 1)$ 递减，$[1, +\infty)$ 递增，求 $y = f(|x-3|)$ 的单调区间。

??? note "解析"
    仍令 $u = |x-3|$。现在 $f(u)$ 的单调性取决于 $u$ 所在的区间：

    - $u < 1$ 时 $f(u)$ 递减；$u \geq 1$ 时 $f(u)$ 递增
    - $u = |x-3| < 1 \Leftrightarrow 2 < x < 4$；$u \geq 1 \Leftrightarrow x \leq 2$ 或 $x \geq 4$

    分区间讨论：

    | $x$ 区间 | $u(x)$ 单调性 | $f(u)$ 单调性 | $y$ 单调性 |
    |----------|--------------|--------------|-----------|
    | $(-\infty, 2]$ | $u \geq 1$，$u(x)$ 递减 | $f(u)$ 递增 | 递减 |
    | $(2, 3]$ | $u < 1$，$u(x)$ 递减 | $f(u)$ 递减 | 递增 |
    | $[3, 4)$ | $u < 1$，$u(x)$ 递增 | $f(u)$ 递减 | 递减 |
    | $[4, +\infty)$ | $u \geq 1$，$u(x)$ 递增 | $f(u)$ 递增 | 递增 |

    **答案**：递增区间 $(2, 3] \cup [4, +\infty)$；递减区间 $(-\infty, 2] \cup [3, 4)$

## 【4】奇函数性质

函数 $f(x)$ 的定义域为 $\mathbb{R}$，若 $f(x+1)$ 与 $f(x-1)$ 都是奇函数，则（ ）

A. $f(x)$ 是偶函数
B. $f(x)$ 是奇函数
C. $f(x) = f(x+2)$
D. $f(x+3)$ 是奇函数

??? note "解析"
    $f(x+1)$ 是奇函数：$f(-x+1) = -f(x+1)$  ①

    $f(x-1)$ 是奇函数：$f(-x-1) = -f(x-1)$  ②

    在①中令 $x \to x+1$：$f(-(x+1)+1) = -f(x+2) \Rightarrow f(-x) = -f(x+2)$

    在②中令 $x \to x-1$：$f(-(x-1)-1) = -f(x-2) \Rightarrow f(-x) = -f(x-2)$

    所以 $f(x+2) = f(x-2) \Rightarrow f(x+4) = f(x)$，即周期为 4。

    由 $f(-x) = -f(x+2)$ 及周期 $f(x+2) = f(x-2)$：

    $f(x+3)$ 是奇函数吗？检验：$f(-x+3) = f(-(x-3))$

    利用结论：由 $f(-x) = -f(x+2)$，令 $x \to x-3$：
    
    $f(-(x-3)) = -f(x-3+2) = -f(x-1)$

    需证 $f(-x+3) = -f(x+3)$。利用周期和奇偶性推导可得 D 正确。

    **验证 D**：由 $f(x+1)$ 奇 $\Rightarrow f(x+1) = -f(-x+1)$，$f(x-1)$ 奇 $\Rightarrow f(x-1) = -f(-x-1)$。

    令 $g(x) = f(x+3)$，则 $g(-x) = f(-x+3)$。

    由 $f(x-1)$ 是奇函数：$f(-(x-4)-1) = -f(x-4-1) = -f(x-5)$。

    又因周期为 4，$f(x-5) = f(x-1)$，且 $f(-x+3) = f(-(x-4)-1) = ...$ 

    最终可推出 $f(-x+3) = -f(x+3)$，即 $f(x+3)$ 是奇函数。

    **答案：D**
